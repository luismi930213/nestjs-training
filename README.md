# Nest Demo
Nest demo application of Luis Sanchez 

## Description

[Nest](https://github.com/nestjs/nest) framework demo of Luis Sanchez for training purpouse.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
